控制塔层-前端Control Tower - Frontend

综合驾驶舱Comprehensive Dashboard

运营KPI总览Operation KPI Overview

Desc: Displays key operational indicator cards, including transport volume, equipment utilization, energy consumption metrics, etc.显示关键运营指标卡片组，包括运输量、设备利用率、能耗指标等

Features: Single-screen overview, highlights key points, supports data drill-down, configurable refresh strategy (near real-time/hourly).一屏全览、重点突出、支持数据钻取、可配置的刷新策略（准实时/每小时）

Effect: Allows management to quickly grasp the overall operational status of the mine and promptly identify indicators deviating from the plan.管理人员能够快速掌握矿山整体运营状态，及时发现偏离计划的指标

任务执行监控Task Execution Monitoring

Desc: Displays a list of currently executing transport tasks and their completion progress.显示当前正在执行的运输任务列表及完成进度

Features: List-based display, progress updates.列表形式展示、进度更新

Effect: Dispatchers can track task completion in real time and handle execution deviations promptly.调度员能实时跟踪任务完成情况，及时处理执行偏差

设备状态分布Equipment Status Distribution

Desc: Displays the quantity distribution of equipment in different statuses using pie/bar charts.以饼图/条形图显示不同状态设备的数量分布

Features: Visual charts (pie/bar), status color coding, drill-down to resource view.可视化图表（饼图/条形图）、状态色彩编码、可钻取到资源视图

Effect: Fleet managers can quickly understand the overall health of the fleet and identify common issues in equipment management.车队经理能快速了解车队整体健康状况，识别设备管理中的普遍性问题

告警与异常中心Alert & Exception Center

Desc: Centrally displays various real-time alerts and exception events.集中展示各类实时告警和异常事件

Features: Alert levels (high/medium/low), real-time push, support for acknowledgment and handling.告警分级（高/中/低）、实时推送、支持确认和处理

Effect: Ensures that critical exceptions are detected and responded to in a timely manner, forming a closed management loop.确保关键异常能被及时发现和响应，形成管理闭环

流程视图Process View

Hauling流程可视化Hauling Process Visualization

Desc: Visualizes the entire Hauling operation process (planning, loading, transport, dumping, return) in the form of Gantt charts or flowcharts.以甘特图或流程图形式，可视化展示Hauling作业（规划、装载、运输、卸载、返回）的全流程

Features: Process segment coloring, key node highlighting, real-time progress updates.流程分段着色、关键节点高亮、实时进度更新

Effect: Managers can intuitively identify process bottlenecks and locate efficiency shortcomings.管理者能直观识别流程瓶颈，定位效率短板

环节KPI监控Stage KPI Monitoring

Desc: Displays core KPI metrics for each stage (e.g., loading, transport) on the process view.在流程视图的每个环节（如装载、运输）上，显示该环节的核心KPI指标

Features: Real-time KPI calculation, association with process nodes, supports YoY/MoM analysis.KPI实时计算、与流程节点关联、支持同比/环比分析

Effect: Helps managers quantitatively assess the operational performance of each stage.帮助管理者量化评估各环节的运营表现

瓶颈分析Bottleneck Analysis

Desc: Automatically identifies and highlights the most time-consuming or longest-waiting stages in the process.自动识别并高亮显示流程中耗时最长或等待时间最长的环节

Features: Automatic bottleneck identification algorithm, historical data comparison, cause drill-down.瓶颈自动识别算法、历史数据对比、原因下钻

Effect: Pinpoints key stages for process optimization and provides data support for decision-making.精准定位流程优化的关键环节，为决策提供数据支持

任务详情追溯Task Details Tracking

Desc: Supports clicking on a task in the process to trace its complete lifecycle and all associated data.支持点击流程中的某个任务，追溯其完整的生命周期和所有关联数据

Features: Full task lifecycle view, drill-down of associated data (vehicle/driver).任务全生命周期视图、关联数据（车辆/司机）钻取

Effect: Provides a complete data chain for troubleshooting and performance analysis of specific tasks.为特定任务的异常排查和绩效分析提供完整数据链条

资源视图Resource View

资源列表与状态Resource List & Status

Desc: Displays a list of all key resources (vehicles, drivers, equipment) and their current status.以列表形式展示所有关键资源（车辆、司机、设备）及其当前状态

Features: Supports multiple resource types, real-time status updates, configurable columns.多资源类型支持、状态实时更新、可配置列

Effect: Provides a unified view of all available resources in the mine area.提供矿区内所有可用资源的统一视图

资源搜索与筛选Resource Search & Filtering

Desc: Provides search and filtering functions based on various conditions such as resource type, name, and status.提供按资源类型、名称、状态等多种条件的搜索和筛选功能

Features: Complex condition search, fuzzy matching.复合条件搜索、模糊匹配

Effect: Helps users quickly locate the specific resources they are concerned about.帮助用户快速定位到其关注的特定资源

资源详情面板Resource Details Panel

Desc: Click on a resource in the list to view its detailed information, associated KPIs, and historical data.点击资源列表中的某个资源，可查看其详细信息、关联KPI和历史数据

Features: Modular information display, historical data charting, links to related information.模块化信息展示、历史数据图表化、关联信息链接

Effect: Provides a 360-degree panoramic view of a single resource.提供单个资源的360度全景视图

资源健康度评估Resource Health Assessment

Desc: Comprehensively displays the health status score (SOH) or maintenance status of resources.综合展示资源的健康状态评分（SOH）或维护状态

Features: Health score model, status light (red/yellow/green) warnings.健康度评分模型、状态灯（红/黄/绿）预警

Effect: Helps managers quickly identify resources that are in a sub-optimal or faulty state.帮助管理者快速识别处于亚健康或故障状态的资源

控制塔层-后端Control Tower - Backend

数据集成架构Data Integration Architecture

实时API接口集成Real-time API Integration

Desc: Obtain data in real-time from the intelligent Hauling system and equipment management system through standardized APIs.通过标准化API从智能Hauling系统和设备管理系统实时获取数据。

消息队列集成Message Queue Integration

Desc: Receive key events and status changes pushed by the application system by subscribing to a message queue.通过订阅消息队列，接收应用系统推送的关键事件和状态变更。

批量数据同步Batch Data Synchronization

Desc: Periodically synchronize historical and summary data from application system databases via ETL processes.通过ETL过程，定期从应用系统数据库同步历史和汇总数据。

数据清洗与转换Data Cleansing and Transformation

Desc: Cleans, transforms format, and standardizes collected raw data before storage.在数据存储前，对采集到的原始数据进行清洗、格式转换和标准化处理。

计算引擎Computation Engine

事件驱动实时计算Event-Driven Real-time Calculation

Desc: Triggers corresponding indicator calculations in real time based on business events.基于从消息队列等渠道获取的业务事件（如装载完成、卸载开始），实时触发相应的指标计算。

流式数据处理Stream Data Processing

Desc: Processes continuous streams of equipment status and location data, e.g., calculating moving average speed.对连续产生的设备状态和位置数据流进行处理，例如计算移动平均速度、判断是否偏离路线等。

实时告警生成Real-time Alert Generation

Desc: Built-in rules engine generates alert information based on preset rules (e.g., waiting timeout, speed anomaly).内置规则引擎，根据预设规则（如等待超时、速度异常）对实时数据进行判断，生成告警信息。

周期性批量计算Periodic Batch Calculation

Desc: Aggregates data within a time period according to a set schedule to generate KPIs like hourly output or daily utilization.按设定的周期（如每小时、每天）对时间段内的数据进行聚合计算，生成中低频KPI，如小时产量、日利用率等。
